# Alpha Biologix Website

## Overview
Build a B2B peptide supplier website for "Alpha Biologix" — a company selling peptides exclusively to doctors and medical practices. Similar style to Pure Health RX but focused specifically on peptides.

## Tech Stack
- Next.js 14+ (App Router)
- Tailwind CSS
- TypeScript
- Lucide React for icons
- Deploy-ready for Vercel

## Brand
- **Name:** Alpha Biologix
- **Tagline:** "Premium Peptides for Medical Professionals"
- **Target:** Doctors, clinics, med spas, telehealth providers
- **Focus:** Peptides ONLY (not prescription drugs)
- **Vibe:** Scientific, premium, trustworthy, modern

## Color Scheme
- Primary: Deep blue/indigo (#4F46E5 to #6366F1)
- Accent: Amber/gold for premium feel (#F59E0B)
- Clean whites and slate grays
- Different from Pure Health RX (which is teal/cyan)
